## 实现内容

### 角色分配权限

- 在角色管理页面中增强了“分配权限”操作。
- 点击“分配权限”后，会在页面下方展开一个静态权限分配区域。
- 没有引入新的 UI 库，也没有使用复杂树组件。
- 保持了现有列表页的结构和视觉风格。

### 展示内容

- 展示当前角色基础信息：
  - 角色名称
  - 角色编码
  - 数据范围
  - 状态
- 展示菜单权限树：
  - 使用 mock 菜单数据
  - 支持目录、菜单、按钮三级结构
  - 使用 `Checkbox` 做前端临时勾选
- 展示权限点列表：
  - 使用 mock 权限点数据
  - 按所属菜单分组展示
  - 使用 `Checkbox` 做前端临时勾选

### 交互边界

- `保存` 仅展示前端成功提示。
- `取消` 会关闭当前权限分配区域。
- 没有真实保存接口。
- 没有复杂半选联动。
- 没有真实权限持久化。
- 没有动态菜单和真实 RBAC 控制。

### 数据层

- 扩展 `src/types/rbac.ts`，新增角色权限详情类型。
- 扩展 `src/mocks/rbac.ts`，新增角色与菜单、权限点的关联数据。
- 扩展 `src/api/rbac.ts`，新增获取角色权限详情的 mock API。
